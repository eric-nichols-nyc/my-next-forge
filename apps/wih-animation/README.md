# WIH Animation App

A Next.js application for creating beautiful animations and interactive experiences.

## Getting Started

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3014](http://localhost:3014) with your browser to see the result.

## Scripts

- `dev` - Start the development server on port 3014
- `build` - Build the application for production
- `start` - Start the production server
- `clean` - Clean build artifacts and dependencies
- `typecheck` - Run TypeScript type checking
